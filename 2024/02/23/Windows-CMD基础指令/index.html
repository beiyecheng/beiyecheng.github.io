Windows CMD 基础命令

1.cmd notion

cd ->means change directory so from here 改变目录

cd .. ->means take me to the parent 返回父录

cd Desktop\Videos 可以直接跳转到Videos

cd ../.. ->means take back two steps

tab键（自动输入）

2.

dir 导航 -> list all directory

dir 目录名/目录名/……（显示指定目录内容）

dir /a（显示隐藏目录内容）

3.

dir *.png -> list all the png

cls ->means clear screen

PgUp go back previous commmands (history)

PgDn go back to the most recent one (foreard)

ipconfig /? ->means access the help and options menu

输入文件名（打开文件）；

4.

mkdir ->means make directory

rmdir ->means remove this directory (删除空目录）

rmdir /s (删除目录以及所有目录中的文件)

ctrl+←键（上一个单词）；

ctrl+→键（下一个单词）；

←键（上一个字符）；

→键（下一个字符）；

5.

cd ‘’C:/……’’（改变目录到绝对路径）；PS：引号不可省略；

输入程序名（打开目录或路径环境变量中的程序）；

path（查看路径环境变量）；

6.

wmic logicaldisk get name（查看驱动器名称）；

c/d/e/f:（打开对应驱动器）；

tree（查看目录结构）；

color（改变界面颜色，具体可查看帮助文件）；

color /? -> help

7.

attrib（更改文件属性）；

8.

del（删除文件）；

echo ******* > 文件名（输入内容至文件内）；

type 文件名（查看文本文件内容）；

echo ******* >> 文件名（添加内容至文件结尾）；

命令 > 文件名（将运行结果输入至文本文件）；

9.

copy 文件名 位置（复制文件）；

xcopy 目录1 目录2 （复制目录1不包括子目录文件至目录2）；

xcopy 目录1 目录2 /s（复制目录1包括子目录文件至目录2）；

move 目录1 目录2 （移动目录1至目录2）；

rename 原目录名 新目录名（重命名目录）；
